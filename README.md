## how to download

- Download git and Node.js
- clone the code ( if you dont know how, ask google )
- open terminal inside FoodIQ folder and download packages by using this command " npm install "
- after this run this command to make your web work " npm run dev "
- go to localhost:3000 in your browser
- have fun bro and dont forget to hit star Btn
